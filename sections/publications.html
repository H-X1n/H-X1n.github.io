核心贡献

  • 提出 以 Mamba-3 为基础的时序投影器（Mamba-3 + SwiGLU），在视觉 token 进入 LLM 前显式完成跨帧时序混合。
  • 通过系统性消融实验确定最优压缩配置，在 MatchTime-Caption 任务上以约 32× token 压缩 保持接近无压缩最优水平。
  • 构建严格的流式因果 QA 协议，约束模型只能访问问题时间戳之前的视频内容。

数据集与训练

  • 覆盖 422 场完整比赛，包含 12 万+ QA 模板与 3.2 万+ 固定 QA。
  • 按 short / mid / long 三级记忆延迟分层，覆盖时间段描述、事件计数、首次事件时间等时序推理任务。
  • 采用四阶段课程训练：视觉语言对齐 → 通用多模态 SFT → 足球领域适配 → 完整比赛流式 QA。

ARRT-DETR: Attention-Residual RT-DETR for Small-Object Detection

第一作者 · 大连交通大学、复旦大学、上海工程技术大学、西交利物浦大学

BMVC 在投 CCF-C 第一作者

面向无人机航拍与密集交通场景的小目标检测问题，重新设计 RT-DETR decoder 的跨层残差信息流，缓解深层 refinement 中弱定位线索被削弱的问题。

方法贡献

  • 提出 ARRT-DETR，在保持 RT-DETR backbone、hybrid encoder 和检测头不变的前提下，将 decoder 中的标准加性残差替换为 Attention Residual。
  • 让每层 decoder 根据当前 query 内容自适应检索并聚合历史 decoder 状态，缓解小目标定位信息在多层迭代中的丢失。
  • 设计 block-wise 近似版本 BARDETR，用于分析跨层历史聚合在精度与效率之间的权衡。

实验结果

在 VisDrone2019 小目标检测基准上，三层 decoder 的 ARRT-DETR 达到 47.9 mAP50 与 29.6 mAP50-95，相较 RT-DETR-L 在参数量、GFLOPs 和延迟更低的情况下取得更高检测精度。
