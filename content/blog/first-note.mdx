一个个人主页不只是简历的网页版。它更像一个长期维护的索引：项目记录你做过什么，论文记录你如何提出问题，博客记录你如何思考和调试。

This first note is intentionally simple. Replace it with a real technical post when you are ready: an implementation diary, a paper reproduction note, or a short explanation of a tool you built.

## 可以替换的内容

- 深度学习分布式训练（DeepSpeed / FSDP）的实践笔记
- Transformers 模型微调和推理优化记录
- 论文阅读笔记和代码复现经验
- 实习项目中的工程复盘

## What to write next

Keep each post focused on one idea. A useful personal technical blog is not a performance; it is a durable workbench.
